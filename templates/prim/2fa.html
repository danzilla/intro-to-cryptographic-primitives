Data Integrity and Authentication Algorithms

These are schemes based on the use of cryptographic algorithms designed to authenticate the identity of entities as well as to used to protect blocks of data, such as messages, from alteration

2FA Two-factor authentication

Two-factor authentication is a security process in which the user provides two means of identification from separate categories of credentials; one is typically a physical/software generated token, such as a card/Secur ID, and the other is typically something memorized, such as a security code.

The parameter it uses is a random key, HOTP, Length parameter and hash algorithm. Once the value is generated it is used as a second peice of Identification.
